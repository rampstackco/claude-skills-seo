# Contributing to claude-skills-seo

claude-skills-seo is a curated SEO-focused subset of the [claude-skills](https://github.com/rampstackco/claude-skills) catalog. Contributions are limited in scope by design.

## Where to contribute

### Skill content changes
All skill content changes should be made to the source repository at [claude-skills](https://github.com/rampstackco/claude-skills). Once accepted there, the change can be re-synced to this SEO subset.

### Adding or removing a skill from the SEO subset
Open a discussion at [claude-skills discussions](https://github.com/rampstackco/claude-skills/discussions) proposing the change. The discussion should include:

- The skill name and what it does
- Why it belongs (or doesn't belong) in an SEO-focused subset
- Which skill, if any, you'd remove to keep the set focused
- Whether it has external MCP dependencies (which would conflict with the standalone-only philosophy here)

### Bug reports for the SEO repo
If you spot a problem with how the SEO repository has copied or referenced a skill (missing files, broken cross-references, README inaccuracies), open an issue here.

## What's out of scope

- New SEO skills not in claude-skills
- Skills that require external MCP setup (Ahrefs, etc.); those belong in the full catalog
- Non-SEO skills beyond the 2 content companions
- Major reorganization of the skill structure
- Plugin packaging for specific Claude Code harnesses
